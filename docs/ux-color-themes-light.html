Dark vs Light
Dark Mode (Linear/shadcn)
  • ✓ Matches developer environments
  • ✓ Reduces eye strain in long sessions
  • ✓ Familiar to target users
  • ✓ Content-focused (less UI glare)
  • ✓ Modern, premium aesthetic
Light Mode (Acctual)
  • ✓ Clean, professional appearance
  • ✓ Better in bright environments
  • ✓ Easier document/text reading
  • ✓ More accessible for some users
  • ✓ Friendlier first impression
